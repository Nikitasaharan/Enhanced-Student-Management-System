# Student Management System – Lab Assignment 3

## Objective
Enhance the Student Management System using:
- Exception Handling  
- Multithreading  
- Wrapper Classes (Integer, Double)

---

## Features
✔ Validates wrong/empty input using `try–catch–finally`  
✔ Uses **custom exception** → `StudentNotFoundException`  
✔ Uses a **loading animation thread** (`Loader` class)  
✔ Uses **wrapper classes & autoboxing**  
✔ Short, clean and modular packages

---

## Folder Structure
src/
├── exceptions/
├── threads/
├── model/
├── service/
└── Main.java

## Key Classes
| Class | Description |
|-------|-------------|
| `Student` | Uses wrapper classes & calculates grade |
| `StudentManager` | Validates data, handles exceptions |
| `Loader` | Multithreading loading animation |
| `StudentNotFoundException` | Custom exception |
